

# Tabular Data Display and Edits

Display select columns, not the entire record set.

Light weight grid with the following features:
- Sort by every column
- Filter by select columns
- Delete icon with correct permissions, confirmation dialog
- Add icon - open create dialog appropriate for the data with form that allows for save or cancel
- Edit icon - open edit dialog with the existing data loaded.  Save or cancel buttons.
- Pagination: 10 records at a time per configuration.  Navigation to move forward, back, end to end.



# Function Icons
Use broadly understood function icons or add, delete, cancel, back, exit etc
